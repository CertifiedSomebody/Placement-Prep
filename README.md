# PlacementPrep

A Python-based placement preparation application built using CustomTkinter.

## Features
- Aptitude quizzes
- Score tracking
- Modern dark UI
- Local progress saving

## Tech Stack
- Python 3.11
- CustomTkinter
- JSON
- SQLite (planned)
