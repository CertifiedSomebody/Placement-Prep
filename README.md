# PlacementPrep

A Python-based placement preparation application built using CustomTkinter.  
The project is designed to help users practice Aptitude, Logical Reasoning, and Programming questions with progress tracking, analytics, and persistent storage.

## Features

- Aptitude, Logical Reasoning, and Programming quizzes
- Timer-based question system
- Score tracking and result evaluation
- Quiz history with timestamps
- Analytics dashboard with performance statistics
- Category-based quiz selection
- Local data persistence using SQLite
- Modern dark-themed UI using CustomTkinter

## Tech Stack

- Python 3.11
- CustomTkinter (UI framework)
- SQLite (database)
- JSON (question storage)
- Matplotlib (data visualization)

## Project Structure

- ui/ : Contains all UI screens (Home, Quiz, Result, Dashboard, History, Sidebar)
- logic/ : Core quiz logic and timer management
- database/ : SQLite database operations
- data/ : Question banks stored in JSON format
- main.py : Application entry point and screen controller
